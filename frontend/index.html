Security

Trust by design.

On chain verification

Every decision hash is written to GuardianRegistry. Independently verifiable by anyone with no trust assumption required.

Machine verifiable audit

Decisions include cryptographic proofs readable by other smart contracts. No off chain intermediaries, no hidden state.

Policy enforcement

Define spending limits, whitelist protocols, and block suspicious patterns. Policies run before execution, not after the damage.

Immutable history

The complete record of every evaluation lives permanently on Pharos. Tamper proof, censorship resistant, always available.
